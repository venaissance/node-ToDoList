# node-gtd

node.js 开发的命令行 GTD 应用

### Windows用户安装指南

```
npm install -g node-gtd
或
yarn add global node-gtd
```

### Mac安装指南

1. 安装命令 `npm install -g node-gtd` 或 `yarn add global node-gtd`
2. 转格式 
  - `brew install dos2unix`
  - `sudo dos2unix 你的全局node_modules路径/node-gtd/cli.js`

### 使用

`dodo` 开启任务模式

`dodo -h` 帮助模式

`dodo add [TASK]` 添加任务

`dodo clear` 清空任务
